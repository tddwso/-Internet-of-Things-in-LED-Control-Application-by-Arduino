# Internet of Things in LED Control Application by Arduino

## 預計完成目標:
以物聯網在LED控制應用。

## 使用環境:

Arduino: 使用 Atmel AVR 單片機，採用開源的軟硬體平台，構建於開源 simple I/O 介面板，並具有使用類似 Java、C 語言的Processing/Wiring 開發環境。
